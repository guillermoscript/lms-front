# 0.4 — Remove forwardRef / Replace useContext with use()

## Summary

The project is on **React 19.2.4** (Next.js 16.1.6). React 19 makes `ref` a regular prop (no `forwardRef` needed) and introduces `use()` as a replacement for `useContext()`. This plan covers both changes.

---

## Part A: forwardRef Removal

### Files Using forwardRef

| File | Category | Uses useImperativeHandle? | Action |
|------|----------|---------------------------|--------|
| `components/teacher/markdown-editor.tsx` | Owned component | Yes | **Refactor** |

Only **1 file** in the entire codebase uses `forwardRef`. No shadcn `components/ui/` files use it (the base-mira variant already uses the React 19 pattern). No files in `lib/` use it.

### Change: `components/teacher/markdown-editor.tsx`

This component uses `forwardRef` with `useImperativeHandle` to expose `insertSnippet()` and `focus()` methods. The single call site (`components/teacher/lesson-editor.tsx`) does **not** pass a ref, so `useImperativeHandle` is currently unused at runtime. However, we should preserve the capability for future callers.

**Before:**
```tsx
import React, { useImperativeHandle, useRef, forwardRef } from 'react'

// ...

const MarkdownEditor = forwardRef<MarkdownEditorHandle, MarkdownEditorProps>(
  ({ id, value, onChange, rows = 12, placeholder, className }, ref) => {
    const taRef = useRef<HTMLTextAreaElement | null>(null)

    useImperativeHandle(ref, () => ({
      insertSnippet(snippet: string) { /* ... */ },
      focus() { /* ... */ },
    }))

    // ... rest of component

    return ( /* ... */ )
  }
)

MarkdownEditor.displayName = 'MarkdownEditor'

export default MarkdownEditor
```

**After:**
```tsx
import React, { useImperativeHandle, useRef } from 'react'

// ...

export default function MarkdownEditor({
  id,
  value,
  onChange,
  rows = 12,
  placeholder,
  className,
  ref,
}: MarkdownEditorProps & { ref?: React.Ref<MarkdownEditorHandle> }) {
  const taRef = useRef<HTMLTextAreaElement | null>(null)

  useImperativeHandle(ref, () => ({
    insertSnippet(snippet: string) { /* ... */ },
    focus() { /* ... */ },
  }))

  // ... rest of component

  return ( /* ... */ )
}
```

Key changes:
1. Remove `forwardRef` import and wrapper
2. Convert to named function declaration with `ref` as a regular prop
3. Remove `.displayName` (named functions have intrinsic display names)
4. Change from `export default` on const to `export default function`

## Part B: useContext to use() Migration

### Files Using useContext

#### Owned Components (recommended to migrate)

| File | Context(s) consumed | Action |
|------|---------------------|--------|
| `components/tenant/tenant-provider.tsx` | `TenantContext` | **Refactor** |
| `components/aristotle/aristotle-provider.tsx` | `AristotleContext` (2 hooks) | **Refactor** |

#### Shadcn / Generated UI Components (skip)

| File | Context(s) consumed | Action |
|------|---------------------|--------|
| `components/ui/sidebar.tsx` | `SidebarContext` | **Skip** - shadcn generated |
| `components/ui/carousel.tsx` | `CarouselContext` | **Skip** - shadcn generated |

#### AI Elements (skip)

16 files in `components/ai-elements/` use `useContext`. These appear to be generated/third-party components and should be **skipped**.

### Change 1: `components/tenant/tenant-provider.tsx`

**Before:**
```tsx
import { createContext, useContext, type ReactNode } from 'react'

// ...

export function useTenant() {
  return useContext(TenantContext)
}
```

**After:**
```tsx
import { createContext, use, type ReactNode } from 'react'

// ...

export function useTenant() {
  return use(TenantContext)
}
```

### Change 2: `components/aristotle/aristotle-provider.tsx`

**Before:**
```tsx
import { createContext, useContext, useState, useCallback, type ReactNode } from 'react'

// ...

export function useAristotle() {
    const ctx = useContext(AristotleContext)
    if (!ctx) throw new Error('useAristotle must be used within AristotleProvider')
    return ctx
}

export function useAristotleOptional() {
    return useContext(AristotleContext)
}
```

**After:**
```tsx
import { createContext, use, useState, useCallback, type ReactNode } from 'react'

// ...

export function useAristotle() {
    const ctx = use(AristotleContext)
    if (!ctx) throw new Error('useAristotle must be used within AristotleProvider')
    return ctx
}

export function useAristotleOptional() {
    return use(AristotleContext)
}
```

## Risk Assessment

**Low risk.** The changes are:
- Syntactic transformations with identical runtime behavior
- `use()` is a drop-in replacement for `useContext()` in non-conditional positions
- The `forwardRef` removal preserves the `useImperativeHandle` pattern (which works with ref-as-prop in React 19)
- Only 3 files need changes total
