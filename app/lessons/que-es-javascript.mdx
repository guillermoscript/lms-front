# ¿Qué es JavaScript?

<Card>
JavaScript (JS) es el lenguaje que añade comportamiento a las páginas web. Mientras HTML define la estructura y CSS el estilo, JavaScript permite: reaccionar a las acciones del usuario, modificar contenido en tiempo real y construir experiencias interactivas.
</Card>

## Probar rápido

Abre las DevTools (F12) y en la consola escribe:

```javascript
console.log('¡Hola, JavaScript!')
```

También puedes usar un archivo `index.html` y un `<script defer>` para ejecutar código después de que cargue el DOM.
